# Customer Retail ML Model Comparison

## Project Overview

This project compares multiple Machine Learning algorithms on a customer retail dataset. The objective is to preprocess customer data, train different classification models, evaluate their performance, and compare the results.

## Features Used

* Quantity
* UnitPrice
* Country

## Machine Learning Models

* Logistic Regression
* Decision Tree Classifier
* K-Nearest Neighbors (KNN)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Google Colab

## Project Workflow

1. Load customer retail dataset
2. Handle missing values
3. Encode categorical features
4. Perform data visualization
5. Split data into training and testing sets
6. Train Logistic Regression model
7. Train Decision Tree model
8. Train KNN model
9. Evaluate using Accuracy Score and Confusion Matrix
10. Compare model performance

## Visualizations

### Quantity vs UnitPrice Scatter Plot

This scatter plot shows the relationship between product quantity and unit price in the retail dataset. It helps identify patterns, trends, and potential outliers present in the data.

### Model Accuracy Comparison

This bar chart compares the accuracy scores of Logistic Regression, Decision Tree, and K-Nearest Neighbors (KNN) models. It provides a visual representation of model performance and helps determine which algorithm performs best on the dataset.

## Results

The performance of Logistic Regression, Decision Tree, and KNN models was compared using accuracy scores. All three models achieved similar performance on the dataset, demonstrating the effectiveness of preprocessing and feature selection.

## Learning Outcomes

* Understanding the Machine Learning workflow
* Data preprocessing techniques
* Model training and evaluation
* Performance comparison of ML algorithms
